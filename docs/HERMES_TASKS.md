5. 完成的任务保留 7 天后归档
6. 🚫 阻塞只阻塞自己，不阻塞其他任务（除非明确依赖）

## 调度规则（cron job 读取此文件）

```
每小时 cron job:
1. 扫描"⏳ 等待中"任务
   → 检查结果文件是否存在
   → 存在则读取结果、更新状态为✅或🚫
   → 结果异常则记录问题，重试或升级为🚫

2. 扫描"🔄 待处理"任务（按优先级）
   → 检查依赖是否满足
   → 依赖满足则激活任务
   → Hermes任务 → 立即执行
   → CC任务 → 委派CC
   → 小Q任务 → 委派小Q

3. 扫描"🚫 阻塞"任务
   → 检查阻塞原因是否解除
   → 解除则移回"🔄 待处理"

4. 不开新任务如果：
   → 有"⏳ 等待中"任务未决（委派的任务还没结果）
   → 有"🔬 进行中"任务未完成（Hermes自己正在做）
   → 有明确依赖当前阻塞任务的🔄待处理任务
   → 普通🚫阻塞任务 → 只阻塞自己，不挡路
```
